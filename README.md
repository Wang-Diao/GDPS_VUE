# GDPS_VUE

##项目描述
采用Java语言， B/S架构，前端使用Vue+ElementUI，后端使用SpringBoot+Mybatis，数据存储在MySQL数据库中。项目分为管理员子系统和用户子系统，管理员子系统包括传感器及数据管理、灾害预测分析、预警信息发布、用户信息管理等模块；用户子系统包括个人信息管理、数据展示查询等模块。

## Project setup

```
npm install
```
